# Guia de instalação

## Pré-requisitos

- Node.js instalado
- npm instalado

## Instalação

No terminal, navegue até a pasta do front:

```bash
cd front/ft_torneio
npm install
```

## Executar o projeto

Depois da instalação:

```bash
npm run dev
```

O Vite irá exibir a URL local do projeto, geralmente:

```text
http://localhost:5173
```

Abra essa URL no navegador para visualizar a aplicação.
